# firewall-based-on-netfilter

开发环境：

操作系统：Ubuntu 14.04.1
内核版本：Linux version 4.4.0-142-generic
开发软件：vim
编译器：gcc version 4.8.4


安装: 
cd mymod

make install

启动:
cd mymod

./shell.o

使用手册:
https://github.com/Cheney0114/firewall-based-on-netfilter/blob/master/mymod/help.txt

卸载:
cd mymod

make uninstall
